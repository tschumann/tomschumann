tomschumann.com
===============

Dependencies
------------

Dependencies are handled by `install.sh`


Developing
----------

After building run `python3 -m http.server` and open http://127.0.0.1:8000/


Releasing
---------

`/install.sh` which will work for a new or existing server.


Installing
----------

```
npm install
node_modules/webpack/bin/webpack.js
```

Testing
-------

```
./test.sh
```
